# BF#
BF# is a brainfuck compiler with more characters to work.

## how to compile
compile it using regular g++ (im using MinGW on windows)
```
g++ main.cpp -o BFSharp.exe
```

# Extension
predefined extensions are:
* .b
* .bf
* .bf#
* .bff
